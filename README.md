
# Freelancer Hours
Freelancer Hours é uma plataforma para que desenvolvedores possam propor horas de trabalho em determinados projetos.
<br/>

### ⚙️ Tecnologias Utilizadas ⚙️

- PHP
- Laravel
- Livewire
- Tailwind CSS
